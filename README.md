<h1> Snake game</h1>

<h5> Here I've Created snake with help of turtle </h5>

<h5>Execute:
  You can excutetpython main.py
   (or)
  In terminal exute sudo ./Sake.sh 
  </h5>

<h3>Controls</h3>
    
    - Top for Up Arrow
    
    - Down for Down arrow
    
    - Right for right arrow
    
    - Left for left arrow
